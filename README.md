### Setup
- `Git Clone https://github.com/RitobrotoMukherjee/upgrade-react-knowledge-year-2025.git`
- `cd upgrade-react-knowledge-year-2025`
- `npm i`
- `npm run dev`
- visit [@localhost](http://localhost:5173/)
